# MP2 Frontend

Este es el frontend del proyecto de **plataforma de streaming**.  
Se encarga de la interfaz de usuario, exploración de películas y conexión con el backend.

---

## Tecnologías

- React + TypeScript + Vite
- SASS
- Fetch API
- Vercel (despliegue)

---

## Scripts

```bash
npm install
npm run dev
npm run build
